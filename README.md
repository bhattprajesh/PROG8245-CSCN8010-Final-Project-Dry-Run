A semantic question-answering system that helps students find campus information
using **Word2Vec** and **GloVe** word embeddings with cosine-similarity retrieval.
---

## ✨ Features

- Semantic search using **Word2Vec** and **GloVe** (50D)
- Text or **voice input** (microphone / audio file upload)
- **Multilingual support** with automatic language detection and translation
- **Text-to-Speech** output for answers
- Two web interfaces: **Gradio**
- Graceful fallback when optional packages are missing
- Top-K retrieval with confidence scoring

## 🚀 Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Train the Word2Vec model

```bash
cd PROG8245-CSCN8010-Final-Project-Dry-Run
python src/train.py
```

This also downloads the Stanford GloVe 6B embeddings (~820 MB) on first run.

### 3. Evaluate retrieval quality

```bash
python src/evaluate.py
```

### 4. Interactive CLI

```bash
python src/predict.py
```

### 5. Launch the Gradio web app

```bash
python app.py
```

Open **http://localhost:7860** in your browser.

The knowledge base (`data/knowledge_base.json`) contains 10 common campus
support questions (library hours, gym, financial aid, etc.) with short,
factual answers.  It is intentionally small to keep the demo self-contained.

---

🧠 How It Works

1. User inputs question via text or voice
2. (Optional) Detect language → Translate to English for retrieval
3. Convert query to vector using GloVe or Word2Vec
4. Compute cosine similarity against knowledge base
5. Return best answer + (optional) translate back and speak it


🛠️ Technologies Used

Embeddings: Gensim (Word2Vec + GloVe)
Web UI: Gradio & Streamlit
Audio: SpeechRecognition, gTTS
Translation: deep-translator + langdetect
Others: NumPy, scikit-learn, NLTK
